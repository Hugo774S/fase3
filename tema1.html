Que es HTML


HTML puede definirse como el lenguaje con el que se define el contenido de las páginas web, es un lenguaje de marcación, el contenido se realiza empleando un conjunto de etiquetas que se usan para definir el texto, para insertar imágenes, cambiar el tipo de letra, tamaño, etc. Para escribir una página web, necesitamos un editor de texto esto es porqué HTML es un lenguaje que se escribe en un documento de texto, entonces el archivo donde está contenido el código HTML es un archivo de texto, que tiene extensión .html. Cuando programemos en HTML lo sé debe programar con un editor de texto, no importa cual sea, lo importante es que todos los trabajos tengan la extensión .html, por ejemplo misitioweb.html



HTML es un Lenguaje de marcado de hipertexto, se utiliza para estructurar y diseñar una página web y todos sus contenidos, contenidos que podrían ser párrafos, una lista con viñetas, imágenes y tablas de datos, entre otros. HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido.
HTML consiste en una serie de elementos que son usados para encerrar diferentes partes del contenido para que se vean de una determinada manera. Las etiquetas de encierre pueden hacer de una palabra o una imagen un hipervínculo a otro sitio, se pueden cambiar palabras a cursiva, agrandar o achicar la letra, etc.
